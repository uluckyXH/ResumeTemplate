# 个人简历网页生成器

这是一个简单的个人简历网页生成器，使用 HTML、CSS 和 JavaScript 构建。它可以根据提供的数据自动生成一个美观的在线简历。

## 如何使用

1. 克隆或下载此仓库到您的本地机器。
2. 打开 `index.html` 文件，找到 `<script>` 标签中的 `resumeData` 对象。
3. 根据您的个人信息修改 `resumeData` 对象中的数据。
4. 在浏览器中打开 `index.html` 文件，查看生成的简历。
